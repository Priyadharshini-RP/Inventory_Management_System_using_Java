# Inventory_Management_System_using_Java

This program is a menu-driven inventory management system developed using Java. It is used to store and manage product details such as ID, name, quantity, and price.

The program uses a HashMap collection to store product objects, where the product ID is used as the key. Each product is represented using a Product class, which contains attributes and a constructor to initialize values.

An auto-increment ID is generated for every new product added to the system. The program provides options through a menu-driven interface, allowing the user to:

Add a new product
Display all products
Exit the program

The user inputs are taken using the Scanner class, and operations are performed using a while loop and switch-case structure.

The addProduct() method is used to insert new products into the inventory, and the displayProducts() method is used to show all stored product details.
